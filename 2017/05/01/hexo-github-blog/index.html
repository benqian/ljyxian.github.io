一、Github配置

创建仓库


登录账号后，点击Github右上方的new repository建立仓库。

在Repository name输入你的仓库名称，如：ljyxian.github.io，然后点击Create repository即可。

添加密钥

在终端输入ssh-keygen -t rsa -C "Github的注册邮箱地址"，然后一直Enter，最后会生成两个文件，id_rsa和id_rsa.pub, 复制id_rsa.pub里面的内容，然后进入ssh设置页面粘贴到key内容框里面，title内容框可以随便填写一个即可。
